cara running:
1. pastikan git sudah terinstal di komputer
2. clone buka terminal kemudian masukkan perintah git clone "url nama reposiroy"
3. kemudian masuk kedalam project yang sudah di clone
4. kemudian install dependecies dengan perintah (npm install)
5. kemudian jalankan project dengan perintan (npm run dev)
